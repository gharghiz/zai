Tech Stack

Python

Backend powered by Python with Flask for the web server and async RSS scraping.

GPT-4o-mini

AI engine for article summaries, sentiment analysis, and market insight generation.

SQLite / PostgreSQL

Dual database support for flexible deployment from development to production.

Telegram Bot API

Automated news delivery with formatted messages, images, and sentiment indicators.

Gunicorn

Production WSGI server for reliable web application hosting.

Feedparser

Robust RSS/Atom feed parsing library supporting 70+ cryptocurrency news sources.

Key Features

Multi-Source Aggregation

Monitors over 70 RSS feeds including CoinDesk, CoinTelegraph, Decrypt, The Block, and Reddit crypto communities for comprehensive coverage.

AI-Powered Analysis

Uses GPT-4o-mini to generate concise summaries, sentiment scores, keyword extraction, and market impact assessments for every article.

Parallel Scraping

Multi-threaded RSS fetching with configurable worker count, timeout handling, and connection pooling for maximum efficiency.

Smart Content Filtering

Automatic deduplication, keyword-based crypto detection, title similarity matching, and quality scoring to ensure relevant content only.

Multi-Language Support

Automatic language detection with support for English and Arabic content, including RTL text rendering for the web dashboard.

Robust Error Handling

Retry logic with exponential backoff for Telegram posting, graceful degradation when AI service is unavailable, and comprehensive logging.
